# Web App

Simple Vercel-ready Next.js version of the MaRe Luxury Growth Engine.

## Includes

- browser-first demo UI
- API routes for snapshot and agent state
- shared domain contracts in `src/lib/types.ts`
- local agent orchestration in `src/lib/agent/`
- demo illustrations and yellow-dot AI marker

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
